VibeChat

VibeChat is a modern real-time chatting and social messaging platform built with a futuristic UI and smooth user experience.

Features

- User Authentication
- Real-time Messaging
- User Search
- Unique User IDs
- Responsive Modern UI
- Dark Theme Interface
- Secure Backend APIs

Tech Stack

Frontend

- React.js
- Tailwind CSS

Backend

- Node.js
- Express.js

Database

- MongoDB


Installation

git clone <repo-link>
cd VibeChat
npm install

Start Frontend

npm run dev

Start Backend

npm start
